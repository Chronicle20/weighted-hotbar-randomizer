Weighted Hotbar Randomizer is a Minecraft mod for Java Edition using the Fabric loader. Key constraints:

- **Client-side only** — no server-side code
- **Fabric mod loader** — uses Fabric API, Mixins, etc.
- **Java** — standard Fabric mod structure with Gradle build
- **GitHub hosted** — will use GitHub Actions for CI/CD

**Why:** The mod's scope is intentionally client-only to keep it simple and compatible.

**How to apply:** All suggestions should respect client-side constraint. Build commands use `./gradlew`. No server-side patterns, no Go/TypeScript/microservice patterns.
